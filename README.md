# Webアプリ最小テンプレート

高校生向けハッカソンの事前学習用サンプルです。
HTML / CSS / JavaScript の役割を確認するための、最小構成のWebページです。

## ファイルの役割

- `index.html`：画面に表示する部品を書くファイル
- `style.css`：色、大きさ、余白などの見た目を書くファイル
- `script.js`：ボタンを押した時などの動きを書くファイル

## 動かし方

一番簡単な方法は、`index.html` をダブルクリックしてブラウザで開くことです。

VSCodeを使う場合は、拡張機能「Live Server」を入れて、`index.html` を右クリックし、`Open with Live Server` を選んでも確認できます。

## 試してほしいこと

### 1. HTMLを変える

`index.html` を開いて、見出しや文章を書き換えてください。

例：

```html
<h1>Webアプリ最小テンプレート</h1>
```

の文字を好きなタイトルに変えてみましょう。

### 2. CSSを変える

`style.css` を開いて、色や大きさを変えてください。

例：

```css
background-color: #2563eb;
```

の色を別の色に変えると、見た目が変わります。

### 3. JavaScriptを変える

`script.js` を開いて、ボタンを押した時に増える数字を変えてください。

例：

```js
count = count + 1;
```

を

```js
count = count + 2;
```

に変えると、ボタンを押すたびに2ずつ増えます。

## 目標

このサンプルを通して、次のことが何となく分かれば十分です。

- HTMLは画面の部品を書く
- CSSは見た目を変える
- JavaScriptは動きを作る
- ソースコードの一部を変えると、画面や動きが変わる
